Before installing Sakura WP, please make sure that following plugins are not installed or deactivated:
- woo thumblog;
- wp-pagenavi.

After uploading the theme to your hosting you are to:
1) Set CHMOD 777 to the folder “your_site/wp-content/themes/sakura/plugins/woo-tumblog/functions/cache”.
2) Set CHMOD 644 to the file “your_site/wp-content/themes/sakura/plugins/woo-tumblog/functions/thumb.php”.
3) The folder “your_site/wp-content/uploads” must be created. Make sure that CHMOD is set to 777.
