Installing All Around Theme

To install this theme you must have a working version of WordPress already installed. For information on installing the WordPress platform, please see the WordPress Codex - http://codex.wordpress.org/Installing_WordPress. Upon downloading the "All Around-allfiles.zip" file, extract the archive and inside you will find full contents of your purchase. The file containing theme files you should upload to your wordpress server is "All AroundWordpressTemplate.zip".

Please Note: You can install the theme in two ways: inside WordPress, or via FTP. If you have trouble installing the theme inside WordPress, please proceed with installing it via FTP and you will be able to get around the issue

Installing from WordPress


Follow the steps below to install inside Wordpress
  • Navigate to Appearance > Themes
  • Click Install Themes and then Upload
  • Navigate to All Around.zip on your computer and click Install Now
  • To activate the newly installed theme navigate to Appearance > Themes select the All Around Theme and press Activate
  • After install you sholud go to section B of this document, continue reading and setup your theme!

Installing via FTP Server


Follow the steps below to install via FTP
  • Unzip All Around.zip to a desired location on your HDD. The extracted files will be uploaded later via FTP
  • Connect to your server via FTP
  • Navigate to /wp-content/themes/ on server
  • Navigate to location of unzipped All Around.zip
  • Upload local folder /All Around/ to remote /wp-content/themes/
  • To activate the newly installed theme navigate to Appearance > Themes select the All Around Theme and press Activate
  • After install you sholud go to section B of this document, continue reading and setup your theme!

Migrating from another theme

If you have already used WordPress with another theme installed and you're migrating to All Around Theme please use the Regenerate Thumbnails plugin to resize all the images uploaded on server. Click the link above and download the free plugin, activate the plugin and regenerate your thumbnails.

Installing Plugins

Upon installation and activation of the All Around Theme you'll be noted to install Custom Plugins used with this theme.

This theme uses the following plugins :

  • All Around Content Slider
  • Revolution Slider
  • WooCommerce Store

Install all the plugins to get all features of this theme. If you're not ready to open a store yet, you can skip WooCommerce Installation. For documentation on these plugins you can read more in the plugin documentation folders.

Upon Plugin installation you'll be noted again to activate the installed plugins.

Setting the Theme Options

After you've done the first two steps of installing the theme and plugins you can continue to Appearance > Theme Options to set your Theme Style and Settings.

General Settings

General Settings include your Theme Color, Logo and Small Logo. Choose any color you want but keep in mind that this color should be vivid for best contrast. Use Upload Logo to change the default AlAround Logo. Click Upload and use WordPress UI to upload or choose your previously uploaded image. If you don’t want to use logo image click Remove and leave the field blank. For best effect use transparent .png images that will blend into your header. This options also work for Small Logo. You can use both logos at the same time or use only one that you prefer.

Header Settings

This section is for setting your header Social Network Profiles. Link your site to your Profiles by entering the Profile URL.

Sidebar Settings

Set your sidebars here. Blog and WooCommerce Archive sidebar switches define should sidebar be used in these Templates. If these sidebars are set ON in Appearance > Widgets new sidebars will appear.

Bellow these switches you have a Sidebar UI for creating new sidebars. To create a new Sidebar that you can use in your pages simply click Add New Sidebar enter title and Save Changes. Once this is done your new Sidebar will appear under Appearance > Widgets and it will be ready to use. Fill it with widgets and override in Sidebar Options on Single Posts. Create as many Sidebars as you need.

Footer Settings

These options define your Footer Widget Areas, Footer Link Icons, the use of Footer Twitter and your copyright information. The Footer Widget Areas is used to define how many columns you have in your footer. One to four are the options. Footer Link Icons are small links in footer slider. The icons should be 60x60px. Footer Twitter is a Twitter Widget just above footer. If you’re using twitter you can enable this option and set username which will be used to retrive the latest tweets. This gadget will automatically change tweets but user interaction is possible. Tweets are changed on user click on this area.

Typography

All Around Theme enables you to use a great number of fonts. More than 500 Google Web Fonts are listed. The options in this section are for choosing the text (main) font and cursive text font. When you select a font from the list it will be shown in the area just bellow the select box. The sizes are fixed because they will work in most cases. If you don’t like the size of your choosed fonts you can alter them via Custom CSS option that is located in the section Advanced Settings.

Contact Settings

These are your contacts, team member profiles. Your Contact Settings / Team Member default options are Name, Picture, Email, Phone, Job, Description and Social Networks. You can add as much contacts as you want. Use Upload to upload image representing the contact and fill in the needed infmoration. To add a new social icon for the contact click Add Network. The input field is the URL of the profile and select box is used to select the appropriate icon. These icons can be replaced with your custom sets that you own. To do this you will need to replace the default icons on your server which are located in wp-content/themes/All Around/images/socialnetworks. Delete the old files or just fill the directory with icons you would like to use. These Contact Settings / Team Memebers are later used in Shortcodes and Shortcode Generator to show your Team Members or to create Contact Forms to contact site personel.

Advanced Settings

These are the advanced options. Responsive means that All Around will adopt a mobile application styled display when viewed on mobiles and handheld devices. If this option is OFF site will render in 1024px wide non-responsive layout that mobile and handheld users can zoom and explore. Select what you prefer. More options aviable are Tracking Code and the Cusotm CSS options. Tracking Code is used to insert scripts into your site header just before the </head> tag. You can use it for Google Analytics or other services that help you track your sites progress. Custom CSS is a special option that should only be used by advanced users to add corrections to the layout that they need without having to alter the theme itself.

By now you have set up your All Around Theme and it’s ready to use! Now you can create some Posts, Pages or WooCommerce Products! Have fun!

Pages

For each page you can edit the default Wordpress Page settings and some extended All Around settings. These settings appear around the Wordpress Post Editor. For some additional settings you can click on the link Screen Options at the top of the Page.

Disable Comments

To disable comments click the Screen Options link at the top of the Page and check the Discussion checkbox. You're new Discussion options will aprear in an Metabox bellow. Uncheck the allow comments check box.

Disable Titles

To disable Page Title turn it OFF in the All Around Page Control Metabox, displayed just bellow the Post Editor, in the Basic Options section. This can be usefull if you wish to create a custom title or on Homepage when you don't want it displayed.

Use Custom Sidebar for a Page

In the All Around Page Control Metabox under Basic Options select which sidebar to use. If you want to create some sidebars you'll need to navigate to Appearance > Theme Options > Sidebar Settings. This process is explained in the Setting the Theme Options in the General section. To use different sidebar with the Page you're editing simply select it from the list displayed.

Sliders

In the All Around Page Control Metabox under Slider Options you can select Slider to use with the Page and setup Slides. Turn the Slider ON by clicking the switch. When Slider is activated new settings will appear. Choose the Slider you want to use. The options are Revolution Slider ( Premium Plugin ) and the two basic styled sliders. For great slides you should use the Revolution Slider. If selected choose the Slider Template from the Revolution Slider select box. If you havn't created any Slide Templates please do so in the Revolution Slider Plugin UI.

Setting up slides for the iCarousel and 3D Slider

iCarousel is another full screen slider but without fancy Revoltion slider animations. For it's Slides you can set the Image to be used as background ( full screen display so choose high resolution images ), and the Title text. You can add links to Title text. For these options you need to click the down arrow in the right corner of the Slide 1. If you wish to add slides click the Add New Slide Button. You can also Save you're slide templates by clicking the Save Slides Template button.

The 3D Slider is a small 3D image carousel. This slider is designed to work with images 480x260px.

Shortcode Generator

In the All Around Page Control Metabox under Shortcode Generator you have a powerfull generator for creating page templates. You should use this insted of manual shortcode inserts to prevent possible errors. You can save the templates you've created, along with their settings so you can use them elsewhere. You can easily create and reuse small templates that you use frequently on your website. To add a new element click the Add New Element button. For element options click the arrow in the right corner of the Element box. Drag and Drop elements is support. Once your element template is finished you can save it by clicking the Save Template button. An icon will appear to input the Save Name. Once saved this Elements Template can be loaded and inserted on all posts by using the icon aside the Template title and then clicking the Insert Content button. If you don’t want to keep the Template anymore click the Remove icon to delete this Template from the database. This is undoable so be carefull what you delete!

Inserting Shortcodes

Once you've created a shortcode layout click the Insert Content button to insert it into Page. When the button is clicked the Shortcode Generator will insert shortcodes at the cursor point in Wordpress Page Editor. Don't forget to update page after this.

Shortcodes

These are the shortcodes that come with the All Around Theme. You can generate these via Shortcode Generator or input them manually.

Accordion

Simple element to create accordions. Insert as many sections as you want, define titles and fill in the accordion content. Content can contain HTML, Embeds, Shortcodes or other elements. When you insert a accordion element the Shortcode Generator will create Shortcodes for it in your layout. You can also insert shortcodes yourself if you prefer it this way. 

// Accordion Shortcode Examples

// [aa_accordion]
// [aa_accordion_section title="$title"] #content [/aa_accordion_section]
// ... as many sections as you need
// [/aa_accordion]
//
// title="$title", section title

//This code will generate an accordion with 2 sections
[aa_accordion]
[aa_accordion_section title="Title 1"] #content 1 [/aa_accordion_section]
[aa_accordion_section title="Title 2"] #content 2 [/aa_accordion_section]
[/aa_accordion]

Blog

The Blog element enables you to insert your blog posts anywhere you want. The blog can be divided up to four columns and in as many rows you need. You can also display only posts from one category and define the order. Ajax load checkbox should be checked if you want the pagination to use Ajax loading of new entries (Ajax means that visitor will not leave the current page while retriveing new posts). You can use as many blog elements in your layout as you want. You can insert blog after blog presenting different categories or styles. When you insert a blog element the Shortcode Generator will create Shortcodes for it in your layout. You can also insert shortcodes yourself if you prefer it this way.

// Blog Shortcode Examples

// [aa_blog type="$number" rows="$number" category="$id" order="$order" pagination="yes" ajax="yes" values="DESC"]
//
// type="$number", type of blog / values supported: 1,2,3,4,5,6 or 7
// rows="$number", number of rows / values supported: 1+
// order="$order", order posts by / values supported: date, title, rand, author, comment_count
// category="$id", category ID / use -1 for all categories, or enter category ID for single category
// pagination="yes", show pagination / values supported: yes, no
// ajax="yes", ajax post loading / values supported: yes, no
// values="DESC", Ascendic posts or Descendic / values supported: ASC, DESC

//This code will generate an one columned Blog element in 5 rows from all categories without AJAX load
[aa_blog type="1" rows="5" ajax="no"]

//This code will generate an four columned Blog element in 1 row from category_id 7 with AJAX load
[aa_blog type="4" rows="1" category="7" ajax="yes"]

Columns

Columns is the basic element for dividing your post into columns. Once inserted each column can also be filled with a Content Template or other elements. When you insert a column element the Shortcode Generator will create Shortcodes for it in your layout. You can also insert shortcodes yourself if you prefer it this way.

// Column Shortcode Examples

// [aa_one_half] #content 1 [/aa_one_half]
// [aa_one_half last="yes"] #content 2 [/aa_one_half]
// 
// last="yes", defines the last column in a row

// This code will generate two columns with 50% in width
[aa_one_half last="no"] #content 1 [/aa_one_half]
[aa_one_half last="yes"] #content 2 [/aa_one_half]

// This code will generate three columns with 33% in width
[aa_one_third last="no"] #content 1 [/aa_one_third]
[aa_one_third last="no"] #content 2 [/aa_one_third]
[aa_one_third last="yes"] #content 3 [/aa_one_third]    

// This code will generate two columns. First 66% in width and the second 33%
[aa_two_third last="no"] #content 1 [/aa_two_third]
[aa_one_third last="yes"] #content 2 [/aa_one_third]    

// This code will generate three columns. First and last will be 25% in width and the middle one 50%
[aa_one_fourth last="no"] #content 1 [/aa_one_fourth]
[aa_one_half last="no"] #content 2 [/aa_one_half]
[aa_one_fourth last="yes"] #content 3 [/aa_one_fourth]

Contact Form

The element used for displaying the Contact Form. Options include preset Contact Settings or Team Members from the All Around Theme Options Page. You can use as many forms in you page as you want. When you insert a contact form element the Shortcode Generator will create Shortcodes for it in your layout. You can also insert shortcodes yourself if you prefer it this way.

// Contact Form Shortcode Examples

// [aa_contactform users="$users"]
//
// users="$users", contact members IDs (preset in Appearence > Theme Options > Contact Options

// This code will generate Contact Form with user ids 1,2,3
[aa_contactform users="1,2,3"]

// This code will generate Contact Form with only user id 1 as option
[aa_contactform users="1"]

Divider and Divider (Transparent)

Divider is a single lined divider of content. The margin can be set and it's set in pixels. Transparent divider is used to align elements without having to use the single lined divider. When you insert a contact form element the Shortcode Generator will create Shortcodes for it in your layout. You can also insert shortcodes yourself if you prefer it this way.

// Divider Shortcode Examples

// [aa_divider margin="$margin"]
//
// margin="$margin", Divider margin

// This code will generate a Divider with margin of 30px
[aa_divider margin="30"]

// This code will generate a Transparent Divider with margin of 10px
[aa_clear margin="10"]

Progress Bar

Progress Bar element gives you a simple, yet very stylish chart element to use with your design. A simple discount sample or productivity chart can add nice detailing to your layout. Set how much and out of, the title and insert into content. These progress bars fill the full width of the page but if you insert them into columns you will get a smaller bar. When you insert a progress bar element the Shortcode Generator will create Shortcodes for it in your layout. You can also insert shortcodes yourself if you prefer it this way.

// Progress Bar Shortcode Examples

// [aa_progressbar howmuch="$number" outof="$number" title="$title]
// 
// howmuch="$number", How much ?
// outof="$number", Out of ?
// title="$title", progress bar title

//This code will generate a progress bar with 80% of the value
[aa_progressbar howmuch="80" outof="100" title="Progress"]

Installing the Demo Content

To install Demo Content navigate to Tools>Import>Wordpress, and install the Wordpress importer if it is not installed. Once installed, upload the All Around XML data provided with the archive downloaded from themeforest.net (remember to unzip the archive XML). When the upload is complete navigate to Appearance>Menus and set the newly created All Around demo menu as the Primary Menu. To install Revolution slider used in the Theme Preview navigate Revolution Slider and create a new slider and name it Homepage. Edit the newly created slider and import the AllAroundRevolutionSlider.txt provided in the Demo Content. Update the slider. After Revolution Slider installation continue to All Around Content Slider Demo intall. Navigate to AllAround Slider and import the AllAroundSlider.json provided in the archive. Now your demo content is installed.
